# lam-keysafe-cli
cli tool for managing all your passwords in one place. 

## Install

```
$ npm i lam-keysafe-cli
```

## Usage

```
$ npx lam-cli <cmd>
```

# help

- `signup`\
signing up with lam-keysafe


- `login`\
login into your lam-keysafe account


- `--version | -V`\
see the current version


- `-help`\
menu for lam-keysafe-cli

## Examples
